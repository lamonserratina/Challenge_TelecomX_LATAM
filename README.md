# 📊 Análisis de Evasión de Clientes — Telecom X

Este proyecto aborda el problema de **churn** (evasión de clientes) en la empresa ficticia **Telecom X**, utilizando técnicas de análisis de datos, limpieza, transformación y visualización. El objetivo es identificar patrones de cancelación y proponer estrategias para mejorar la retención.

---

## 🧠 Objetivo

- Comprender los factores que influyen en la evasión de clientes.
- Transformar datos crudos en información estratégica.
- Proponer recomendaciones basadas en evidencia para reducir el churn.

---

## 📁 Estructura del Notebook

El análisis se desarrolla en un notebook de Google Colab e incluye:

1. **Introducción**: Contexto del problema y objetivos del análisis.
2. **Limpieza y Tratamiento de Datos**: Proceso ETL completo desde archivo `.json` alojado en GitHub.
3. **Análisis Exploratorio (EDA)**: Visualizaciones, segmentaciones y arquetipos simbólicos.
4. **Conclusiones e Insights**: Principales hallazgos del comportamiento de los clientes.
5. **Recomendaciones Estratégicas**: Acciones sugeridas para mejorar la retención.
